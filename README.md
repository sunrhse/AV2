## Como executar o projeto

Clone o repositório:

```bash
git clone COLOQUE_O_LINK_DO_REPOSITORIO_AQUI
```

Entre na pasta do projeto:

```bash
cd NOME_DA_PASTA_DO_PROJETO
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Acesse no navegador:

```txt
http://localhost:5173
```
